macOS Installation Guide

Follow these steps to install ListForge on macOS.

1 Download the Installer

Click the "Download for Mac" button on the main page to get the installer (.dmg) file.

2 Open the DMG File

Double-click the downloaded .dmg file to open it.

3 Drag to Applications

Drag the ListForge app icon into your Applications folder.

4 Open ListForge

Open ListForge from your Applications folder. If you see a security warning, click "Open" to confirm.
